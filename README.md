# Omega Watches Website Clone

A frontend clone of the luxury Omega Watches website built to practice modern web design principles using pure HTML and CSS.

## 🚀 Features
- 📱 Fully Responsive Design (Mobile, Tablet, and Desktop friendly)
- 💎 Luxury UI with elegant typography and clean layouts
- 🧭 Interactive Navigation Bar and Hover Effects
- 📸 High-quality image grids showcasing watch collections

## 🛠️ Tech Stack Used
- **HTML5:** For semantic web structuring
- **CSS3:** For custom styling, Flexbox, CSS Grid, and media queries
